# Animatronic Eyes v1

A 3D-printed animatronic eye system combining mechanical design, servo actuation, embedded programming, and computer vision.

The project uses an open-source eye mechanism as a foundation and extends it with custom electronics, control software, and Raspberry Pi integration.

## Features

- Independent horizontal and vertical eye movement
- Servo-controlled blinking eyelids
- Analog joystick control
- Raspberry Pi integration
- harcascade xml based face tracking
- Real-time servo actuation
- Modular mechanical design for future upgrades

## Hardware

- Arduino
- Raspberry Pi
- SG90 Micro Servos
- 3D Printed Components
- Joystick Module

## Software

- Arduino C++
- Python
- OpenCV
- Servo Library

## Computer Vision Development

A Raspberry Pi + OpenCV face tracking system was successfully developed and tested separately.

Future work includes integrating the computer vision pipeline directly with the animatronic eye mechanism for autonomous tracking.

## Future Development

- Full 2-axis eye mechanism
- Head pan and tilt
- Jaw actuation
- Facial expressions
- AI-based interaction system
- Voice output and environmental awareness
